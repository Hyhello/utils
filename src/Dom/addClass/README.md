#### #addClass

```javascript
utils.addClass(el, cls);
```

给指定 HTML 元素（el）添加 className（cls）

- **引入版本**

    1.0.0

- **参数**

    1. el (HTMLElement): 待处理的 HTML 元素。
    2. cls (string)：待添加的 className。

- **返回**

    (void): 无返回值。

- **例子**

```javascript
utils.addClass(document.body, 'aaa');
// => void
```
