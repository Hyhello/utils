#### #isLeapYear

```javascript
utils.isLeapYear(year);
```

检测变量（year）是否是闰年。

- **引入版本**

    1.0.0

- **参数**

    1. year (number): 待检测的变量。

- **返回**

    (boolean): 如果 year 是闰年，则返回 true，否则返回 false。

- **例子**

```javascript
utils.isLeapYear(2020);
// => true

utils.isLeapYear(2021);
// => false
```
